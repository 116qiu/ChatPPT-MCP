# chatppt-mcp
基于chatppt进行的AI PPT生成服务，可以满足基于主题或者要求、以及上传文档进行生成ppt，支持在线编辑与下载。

目前已经逐步开放ChatPPT-Server 对应的STDIO模式，即将也会开放对应的SSE协议。
